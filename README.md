# Long life;

Spring Boot, React, Python(FastAPI), MariaDB를 기반으로 한 **운동 추천 & 기록 관리 서비스**입니다.  
사용자의 운동 목표와 기록, 친구 관계, 스토리(피드)를 통합 관리하고,  
추후 AI Agent를 통해 **맞춤형 운동 추천**과 자동화된 피드백을 제공하는 것을 목표로 합니다.

---

## 1. 프로젝트 개요

- **목표**
  - 요즘같은 100세 시대에 사용자가 자신의 운동 목표와 일정을 쉽게 관리하고,
  - 친구들과 운동 스토리를 공유하며, 건강하게 운동할 수 있게끔 도와주는 서비스 웹사이트
  - AI Agent를 통해 개인 맞춤형 운동 추천을 받을 수 있는 서비스 구현

- **핵심 컨셉**
  - **회원 → 목표/운동일지/스토리/친구 + AI Agent**
  - 소셜 기능(스토리, 친구, 좋아요/댓글)을 결합한 운동 동기부여 플랫폼
  - 이후 Physical/Health 관련 AI 추천 모델로 확장 가능한 구조

---

## 2. 기술 스택

- **Backend**
  - Java, Spring Boot
  - MariaDB (RDBMS)
- **Frontend**
  - React
- **AI / Agent**
  - Python, FastAPI
  - Gemini 기반 AI Agent (Google ADK)
- **협업 / 기타**
  - GitHub 기반 형상관리
  - Figma (UX/UI, 유스케이스 다이어그램, 와이어프레임 설계)

---

## 3. 주요 기능 (도메인 기준)

### 3-1. 회원 / 고객

- 회원가입, 로그인, 프로필 관리
- 기본 역할: **고객 → 회원 → (스토리 / 목표 / 운동일지 / 친구 / AI Agent)**

### 3-2. 스토리(피드)

- 스토리 작성 / 수정 / 삭제 (CRUD)
- 좋아요, 댓글/답글 기능
- 공유(수) 집계

### 3-3. 목표 관리

- 운동 관련 **목표 설정**
  - 체중, 운동 빈도 등
- 목표 달성 여부(성공 / 실패) 기록
- 목표 유형: 유산소 / 무산소 등으로 확장 예정

### 3-4. 운동일지

- 캘린더 기반 운동일지 관리
  - 날짜별 운동 기록 CRUD
  - 운동 내용/시간/강도 기록
- 운동일지 공유 기능 (선택)

### 3-5. 친구 기능

- 친구 요청 / 수락 / 거절
- 친구 목록 / 검색
- 친구의 스토리/운동일지 열람 권한 관리

### 3-6. 관리자

- 회원 관리
  - 회원 정보 수정, 정지/삭제 등
- 등록 관리
  - 운동/카테고리/기본 데이터 CRUD
- 스토리 관리
  - 문제 게시글 가리기/삭제 등 콘텐츠 관리 기능

### 3-7. AI Agent

- 초기: 챗봇 형태의 운동 Q&A / 동기부여 메시지 제공
- 이후:
  - 사용자의 목표/운동일지/기초정보(DB)를 기반으로 한 **맞춤형 운동 추천**
  - 세션 메모리를 활용한 개인별 대화 맥락 유지
  - 장기적으로 Vector DB를 활용해 Physical/Health 추천 모델로 확장

## 5. 향후 개발 계획 (Roadmap)

### Phase 1 – 기본 서비스 MVP

- 회원가입 / 로그인
- 운동일지 캘린더 CRUD
- 목표 설정 및 진행상황 표시
- 기본 스토리 작성 및 피드 보기
- 친구 요청/수락/목록 기능

### Phase 2 – 소셜 & 운영 기능 강화

- 스토리 좋아요 / 댓글 / 공유수 카운트
- 관리자 페이지(회원/스토리 관리)
- 운동일지/목표 통계 대시보드(간단 시각화)

### Phase 3 – AI Agent 연동

- FastAPI 기반 AI Agent 서버와 웹 서비스 연동
- 사용자 프로필 + 운동일지 + 목표 데이터를 기반으로:
  - 오늘의 운동 추천
  - 주간/월간 운동 계획 제안
- 세션 메모리를 이용한 개인별 맞춤 대화 흐름 유지

### Phase 4 – 고도화

- 추천 로직 고도화(룰 기반 → 통계/ML 기반으로 확장 가능)
- 장기적으로 Vector DB를 활용해 Physical/Health 관련 외부 데이터와 연동하여  
  더 정교한 맞춤형 운동 추천 시스템으로 확장
- 추가적인 기능(카카오 맵 API를 이용한 지도 UI, 운동 애니메이션 UI 등) - 각자 기능들 Level Up
